Onboarding

What is OpenClaw? 🦞

An open-source personal AI platform that runs on your machine and connects to everything.

🧠

Skills-First

Define capabilities as markdown files. Your AI learns by reading SKILL.md files.

🤖

Multi-Agent

Run multiple specialized agents — each with their own personality and workspace.

💬

Multi-Channel

Telegram, Discord, web UI — talk to your agents from anywhere.

🔓

Open Source

68k+ GitHub stars. MIT licensed. Runs locally or on any server.

Your AI, supercharged with superpowers — skills you define, tools you connect, that can run 24/7.

For those new to OpenClaw — it's an open-source AI platform. Think of it as your AI, supercharged with superpowers — skills you define, tools you connect, that can run 24/7.

Configure multi-agent orchestration and delegation rules.

To get started you need a server — a VPS works great. Hostinger has a one-click setup for OpenClaw. Then it's 3 commands: install, onboard, start. The key files: SKILL.md teaches abilities, SOUL.md defines personality, AGENTS.md sets up multi-agent orchestration.
Core Concepts

Skills & Agents

Skills = Abilities

# ~/.openclaw/skills/pr-review/SKILL.md

# PR Review Skill
Review pull requests with focus on:
- Code quality and patterns
- Security vulnerabilities
- Test coverage

## Steps
1. Fetch PR diff via `gh pr diff`
2. Analyze each changed file
3. Write structured assessment
4. Post review comments

Agents = Specialists

# openclaw.json → agents.list

{
  "id": "software-engineer",
  "name": "Ciel",
  "model": "gpt-5.3-codex",
  "identity": {
    "name": "Ciel",
    "theme": "Moonlit orchestrator",
    "emoji": "🌙"
  },
  "tools": { "profile": "full" }
}

OpenClaw comes with built-in skills you can enable during onboarding — favorites: agent-config and skill-creator. You can also create your own skills. Browse more on ClawHub 🏪

Skills are markdown files that teach your agent what to do. Agents are configured in JSON with their own model, personality, and tools. OpenClaw comes with built-in skills you can enable during onboarding — agent-config and skill-creator are the best to start with. You can also create your own skills and browse community skills on ClawHub.

The Coding Buddy Setup

A simple, workable architecture anyone can start with.

Client
📱 Telegram / Discordyour messaging app
OpenClaw Gatewayreceives & routes messages
message routed to your agent
Brain
🤖 Your AgentGPT-5.x / Claude (OAuth)
📝 SKILL.md + SOUL.mdreads skills & personality
reads project, writes code, runs tests, commits
Exec
git pull~/Projects/*
Read & Codeproject resources
Test & Commitlint, test, push
gh pr create+ screenshots
result delivered back via your channel
Output
✅ PR Ready for Reviewcode diff, screenshots, description
📱 Review on your phoneapprove / request changes
Any VPSHardware
~$20/moSubscription only
~2 minEnd to end
Here's the simple setup anyone can start with. Message enters via Telegram or Discord, hits the Gateway, routes to your agent. The agent reads your skills, reads the project, writes code, tests, commits, pushes, creates a PR. Result flows back to your phone. Any VPS, about $20/mo, 2 minutes end to end.
